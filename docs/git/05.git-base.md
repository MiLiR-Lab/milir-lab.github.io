## 1. 查看版本

```
git --version
```

## 2. 配置用户信息

```
git config --global user.name "Your Name"
git config --global user.email "you@example.com"
```

## 3. 创建仓库

```
git init
```

## 4. 添加文件

```
git add <file>
```

## 5. 提交更改

```
git commit -m "Commit message"
```

## 6. 查看状态

```
git status
```

## 7. 查看历史

```
git log
```

## 8. 创建分支

```
git branch <branch-name>
```

## 9. 切换分支

```
git checkout <branch-name>
```

## 10. 合并分支

```
git merge <branch-name>
