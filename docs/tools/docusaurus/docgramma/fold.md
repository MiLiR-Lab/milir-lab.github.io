## 文档折叠

使用 `details` 和 `summary` 标签来创建折叠。

<Tabs>
<TabItem value="markdown" label="实现语法">
```
<details>
  <summary>折叠标题</summary>
  <p>折叠内容</p>
</details>
```
</TabItem>
<TabItem value="preview" label="效果预览">

<details>
  <summary>折叠标题</summary>
  <p>折叠内容</p>
</details>

</TabItem>
</Tabs>
