## 文档折叠

使用 `Collapsible` 组件来创建带最小化/最大化按钮的折叠窗口。

<Tabs>
<TabItem value="markdown" label="实现语法">

```jsx
<Collapsible title="折叠标题">
  <p>折叠内容</p>
</Collapsible>
```

</TabItem>
<TabItem value="preview" label="效果预览">

</TabItem>
</Tabs>
