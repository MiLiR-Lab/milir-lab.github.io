title: 文档表格
---

- 使用 `|` 符号分割每列
- 使用 `-` 符号来创建表头
- 使用 `:` 符号来控制对齐

## 表格:左对齐

将 `:` 符号放到 `-` 符号左边表示左对齐。

<Tabs>
<TabItem value="markdown" label="实现语法">
```
|   1-1-1   |   1-2-1   |   1-3-1   |   1-4-1   |
|   :-      |   :-      |   :-      |   :-      |
|   2-1-1   |   2-2-1   |   2-3     |   2-4-1   |
|   3-1-1   |   3-2     |   3-3-1   |   3-4-1   |
|   4-1     |   4-2-1   |   4-3-1   |   4-4-1   |
```
</TabItem>
<TabItem value="preview" label="效果预览">
|   1-1-1   |   1-2-1   |   1-3-1   |   1-4-1   |
|   :-      |   :-      |   :-      |   :-      |
|   2-1-1   |   2-2-1   |   2-3     |   2-4-1   |
|   3-1-1   |   3-2     |   3-3-1   |   3-4-1   |
|   4-1     |   4-2-1   |   4-3-1   |   4-4-1   |
</TabItem>
</Tabs>
