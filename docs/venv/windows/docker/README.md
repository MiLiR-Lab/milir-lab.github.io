Docker 是一个开源的容器化平台，用于开发、部署和运行应用程序。

:::info
- Docker 官网： [Docker](https://www.docker.com/)

Docker 允许开发者将应用及其依赖项打包成一个轻量级、可移植的 容器，确保应用在不同环境中一致运行。
:::
