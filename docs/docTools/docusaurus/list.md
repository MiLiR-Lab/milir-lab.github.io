## 无序列表

使用 `-`、`*` 或 `+` 符号来创建无序列表。

### 方法一

使用 `-` 符号来创建无序列表。

<Tabs>
<TabItem value="markdown" label="实现语法">
```
- 列表项
- 列表项
  - 子列表项
  - 子列表项
```
</TabItem>
<TabItem value="preview" label="效果预览">
- 列表项
- 列表项
  - 子列表项
  - 子列表项
</TabItem>
</Tabs>

### 方法二

使用 `*` 符号来创建无序列表。

<Tabs>
<TabItem value="markdown" label="实现语法">
```
* 列表项
* 列表项
  * 子列表项
  * 子列表项
```
</TabItem>
<TabItem value="preview" label="效果预览">
* 列表项
* 列表项
  * 子列表项
  * 子列表项
</TabItem>
</Tabs>

### 方法三

使用 `+` 符号来创建无序列表。

<Tabs>
<TabItem value="markdown" label="实现语法">
```
+ 列表项
+ 列表项
  + 子列表项
  + 子列表项
```
</TabItem>
<TabItem value="preview" label="效果预览">
+ 列表项
+ 列表项
  + 子列表项
  + 子列表项
</TabItem>
</Tabs>

## 有序列表

使用数字和 `.` 符号来创建有序列表。

<Tabs>
<TabItem value="markdown" label="实现语法">
```
1. 列表项
2. 列表项
   1. 子列表项
   2. 子列表项
```
</TabItem>
<TabItem value="preview" label="效果预览">
1. 列表项
2. 列表项
   1. 子列表项
   2. 子列表项
</TabItem>
</Tabs>

## 任务列表

使用 `- [ ]` 或 `* [ ]` 来创建任务列表。

### 方法一

使用 `- [ ]` 来创建任务项。

<Tabs>
<TabItem value="markdown" label="实现语法">
```
- [ ] 任务项
- [x] 任务项
- [ ] 任务项
```
</TabItem>
<TabItem value="preview" label="效果预览">
- [ ] 任务项
- [x] 任务项
- [ ] 任务项
</TabItem>
</Tabs>

### 方法二

使用 `* [ ]` 来创建任务项。

<Tabs>
<TabItem value="markdown" label="实现语法">
```
* [ ] 任务项
* [x] 任务项
* [ ] 任务项
```
</TabItem>
<TabItem value="preview" label="效果预览">
* [ ] 任务项
* [x] 任务项
* [ ] 任务项
</TabItem>
</Tabs>
